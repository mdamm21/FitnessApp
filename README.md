# AI Fitness App

Das Ziel ist die Entwicklung einer AI Fitness App, welche mithilfe von KI individuelle Trainigspläne für den User generiert.
---

## Features

- **Personalisierte Trainingspläne**  
  Automatisch generierte Workouts basierend auf:
  - Fitnesslevel  
  - Trainingsziel (Muskelaufbau, Abnehmen, Gesundheit, Ausdauer, etc.)  
  - Verfügbarkeit (Tage/Woche, Dauer pro Einheit)  

- **KI-gestütztes Coaching**  
  - Generierung von Übungen inkl. Sätzen, Wiederholungen & Pausen  
  - Dynamische Anpassung basierend auf deinem Fortschritt  
  - Optionale Form- und Technik-Hinweise (z. B. „Rücken gerade halten“)

- **Tracking & Fortschritt**  
  - Übersicht deiner Workouts  
  - Verlauf für Volumen, Trainingstage und ggf. Gewicht  
  - Motivation durch Badges und Meilensteine  

- **Offline-freundlich**  
  - Workouts können lokal gespeichert werden  
  - Synchronisation, sobald wieder Internet verfügbar ist  

- **Datenschutz im Fokus**  
  - Datenschutzkonformität nach DSGVO
  - mit Firebase DB
  - Klare Trennung von persönlichen Daten und Trainingsdaten  

## Stand des Projekts
- login und speicherung der Daten funktioniert einwandfrei
- Trainingspläne werden von 3 hinterlegten anhand der Eingaben ausgewählt
- AI wird zu späterem Zeitpunkt implementiert
